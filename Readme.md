## Project Structure 📂

- **Backend** → Node.js + Express + MongoDB  
- **Frontend** → React.js  

---

## How to Run the Project ⚡

1. **Clone the repository**
   ```bash
   git clone https://github.com/Pratham00007/Restrauant_Booking_MERN.git
   cd Restrauant_Booking_MERN
   ```

2. **Run the Backend first**
   ```bash
   cd backend
   npm install
   npm run dev
   ```

3. **Run the Frontend**
   ```bash
   cd ../frontend
   npm install
   npm run dev
   ```

4. Open your browser and navigate to:
   ```
   http://localhost:3000
   ```
